# Vue+Express 前后端脚手架

> 构建以 Vue 作为客户端，Express 作为服务端的全栈独立项目脚手架

## Router

- `/`：项目客户端 UI
- `/api`：服务端数据
- `/server`

## Menu

```bat
└─src
    ├─client
    │  ├─public
    │  └─src
    │      ├─assets
    │      └─components
    └─server
        ├─bin
        ├─public
        │  ├─images
        │  ├─javascripts
        │  └─stylesheets
        ├─routes
        └─views
```
